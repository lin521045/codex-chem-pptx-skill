# 视觉 QA 提示模板

将 slide 渲染为单页图片后，使用下面这段检查提示。

```text
请作为第二轮化学与化工 PPT 视觉审查者，逐页检查这些幻灯片图片，默认认为一定存在问题。

重点寻找：
- 谱图、流程图、结构式或设备图缺失
- 化学式上下标显示错误
- 中文与英文术语之间缺少空格
- 图题、坐标轴、峰指认、工艺条件被裁切
- PFD / P&ID 图框越界或过小
- 机理箭头、步骤编号、注释之间互相遮挡
- 表格密度过高导致字号过小
- 风险矩阵、SOP 步骤框和正文不对齐
- 残留泛化占位词，如“插图”“图片”“图表”
- 残留 placeholder 文本

请按页输出：
1. 发现的问题
2. 严重程度
3. 推荐修复方向

不要因为内容“技术味很强”就放松审查；技术页面更容易藏排版问题。
```
