# 化学与化工首份 PPT 工作流模板

这是给 Codex 直接使用的首轮工作流模板，适合“我有实验手稿、工艺 brief 或培训资料，请你先做出第一版化工 PPT”这类任务。

## 使用方式

1. 先准备一份 brief，可参考 `source-brief-template.zh.md`。
2. 把 brief、论文手稿、流程说明或培训材料贴给 Codex。
3. 再附上下面这段任务模板。

## 模板任务

```text
请使用 $chem-pptx-codex 完成这份化学与化工 PPT 任务。

目标：
- 基于我提供的资料，生成一份结构完整、术语专业、以简体中文为主但自然保留英文化学术语的 PPT 初稿。

请按以下流程工作：
1. 先判断场景属于：
   - 学术研究 / 实验报告
   - 化工工艺设计
   - 安全生产培训
2. 按对应的默认大纲生成页结构。
3. 全文执行以下预处理：
   - 中文与英文 / 数字之间保留半角空格
   - 单位统一为 mol/L、kJ/mol、m³/h、wt% 等规范格式
   - 化学式和同位素尽量输出为 H₂O、SO₄²⁻、¹³C NMR 这类 Unicode 形式
4. 每一页都加入与内容匹配的专业图示占位符，例如：
   - [在此插入 ChemDraw 分子结构图]
   - [在此插入 Aspen PFD/P&ID 流程图]
   - [在此插入 NMR/IR/MS 谱图]
   - [在此插入 HAZOP 风险矩阵]
5. 如果我提供了现成模板，请优先走模板编辑路径；如果没有，请用 PptxGenJS 从零创建。
6. 完成后执行 QA：
   - 检查占位符残留
   - 检查中英混排空格
   - 检查化学式上下标和单位
   - 检查视觉拥挤、图题裁切、流程图越界等问题

输出要求：
- 先给出建议目录
- 再给出配色和字体方向
- 说明每页的布局类型和图示占位符
- 产出可继续迭代的 deck 文件或生成脚本
- 最后说明做了哪些 QA 检查
```
